Face Generator

p5.js

An interactive random face generator with mouse-tracking eyes and simple key controls that change expressions, backgrounds, and allow saving snapshots.

Sketch

Reflection

My idea for this project was to make a playful face generator that feels alive and interactive. I separated the code into functions for each part of the face – head, eyes, nose, and mouth – so I could focus on them one by one and easily make changes. Random ranges control the size and color, so every new face looks a little different. I also added simple interactions: clicking changes the background, pressing R makes a new random face, M changes the mouth expression, and S saves the image.

While building it, I learned how important small details are. Making the pupils follow the mouse with map() gave the face more personality, but only when I kept the range small enough. I struggled with keyPressed() not working at first, and fixed it by cleaning up the conditions and testing core interactions with console.log. I also had to be careful about drawing order: the instruction text needed to be drawn after pop(), otherwise it moved off the screen.

One limitation is that I couldn’t implement hair. I wanted to generate hairstyles that adapt to each head shape, but it was harder than I expected. Since every head has a slightly different width and height, the hair needed to respond dynamically, and I wasn’t sure how to calculate it cleanly. With more time, I would explore arcs, curves, or separate hair assets that scale based on the head size.
