Exquisite Corpse

A hybrid character built from three partners’ drawings, featuring animated ears and clickable belly patterns inside a Halloween-themed scene.

Sketch

Process & Reflection

I started with my friend’s dog Choco, who acts so human that I imagined “Choco commuting to work.” Sanjana shared a cat-inspired character; I especially liked the belly pattern in her sketches. Matthew drew a scarecrow wearing a pumpkin Halloween costume. To harmonize our parts, I used my head, Sanjana’s cat belly translated into a four-ellipse pattern, and Matthew’s scarecrow legs and boots. I set the background to a Halloween color gradient and added a corner spiderweb so Matthew’s theme carries through the whole scene.

I created three belly patterns and used an if/else structure to switch between them. On each click, patternMode updates so the pattern cycles. I struggled with the ear animation at first because the rotation pivot felt wrong, but fixing the pivot point and using translate() → rotate() in that order made the flap feel natural. Then I used sin(frameCount * flapSpeed) to animate the ears.

I split the code into user-defined functions like drawBody(mode), drawHead(flapSpeed, flapDeg), and drawFeet(), and passed real values from draw(). For the background, I used a simple loop to make a vertical gradient and more loops to draw the spiderweb. If I had more time, I would add a small “boot bounce” animation so the character feels even more alive.
